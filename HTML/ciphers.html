Ciphers

nCode supports a range of both historic and custom-made ciphers. Each one transforms text in a unique way — try them, learn how they work, or create your own.

  • Caesar Cipher

    🔁 Simple letter shift

    Each letter is shifted by a fixed number (e.g., +3 → A → D).

    Very intuitive and iconic.

  • ROT13

    Fixed Caesar Shift

    🔄 A Caesar cipher with a fixed shift of 13.

    Often used for mild obfuscation on the internet.

  • Atbash Cipher

    🔍 Letter mirroring

    Replaces each letter with its opposite in the alphabet (A ↔ Z).

    Simple and ancient.

  • Monoalphabetic Substitution

    Custom alphabet

    Each letter is replaced with a unique substitute.

    Good for learning about frequency analysis.

  • Vigenère Cipher

    🧬 Keyword-based shifting

    Uses a repeating keyword to shift letters in multiple alphabets.

    Was considered unbreakable for centuries.

  • Affine Cipher

    Mathematical encoding

    Applies a formula: (a×x + b) mod 26 to each letter's position.

    Combines math and substitution.

  • Keyword Cipher

    🗝 Alphabet reordered by keyword

    Uses a keyword to create a custom alphabet.

    A structured twist on monoalphabetic ciphers.

  • Rail Fence Cipher

    Zig-zag pattern

    Letters are written in a zigzag pattern and read line by line.

    Simple and visual transposition cipher.

  • Columnar Transposition

    ▦ Grid-based reordering

    Write the message in a table by rows and read columns in a keyword order.

    Teaches structural transformation of data.

  • Mirror Text

    🔁 Reverse text

    Reverses the message left to right.

    Simple, fun, and often used for obfuscation.

  • Scytale Cipher

    📜 Ancient transposition

    Wraps a strip of text around a rod to reveal the message.

    Used by ancient Spartans — easy to simulate with row-column logic.

  • Custom Cipher

    🧩 User-defined rules

    Allows users to create their own substitution or transposition rules.

    Flexible and great for learning.
